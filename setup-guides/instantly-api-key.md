# How to Get Your Instantly.ai API Key

## Step 1 — Open Instantly Settings

1. Log into [instantly.ai](https://instantly.ai)
2. Click your avatar (bottom-left) → **Settings**
3. Go to **Integrations** → **API**

## Step 2 — Copy Your API Key

Your API key is displayed on this page. Copy it — it looks like a long alphanumeric string.

## What Gets Pulled

- All campaign names and statuses
- Emails sent (7-day and 30-day windows)
- Open rates and reply rates
- Campaign-level analytics
